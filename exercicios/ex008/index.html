Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>(não semântico).

Nesta, eu tenho uma termo em destaque usando a tag<strong>(semântico)

Itálico / Ênfase

Nesta frase tenho um termo em itálico usando a tag <i> (não semântico).

Nesta frase tenho um termo em ênfase usando a tag <em> (semântico).

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

Texto grande e texto pequeno

Podemos tambem criar textos grandes, sendo este usando a tag <big>, porem depreciada, nao faz sentido usar mais essa tag, agora podemos fazer isso usando as CSS.

e textos pequenos, usando a tag<small>, que ainda continua sendo usada, por conter alguma ênfase, significando alguma coisa.

Texto deletado e Texto inserido

Podemons escrever textos deletados tipo assim, para dizer que ele pode ser lido, mas não considerado, usando a tag <del>

E também podemos escrever um texto inserido tipo este, para dizer que ele foi escrito depois, usando a tag <ins>

da pra se usar também a tag <u>, para fazer ficar sublinhado, porem não e semantico, enquanto a tag <ins> tem algum significado semantico.

Texto sobrescrito e texto subscrito

Usados para apresentar coisas do tipo x25 + 3, usando a tag <sup>.

E coisas do tipo H2O, usando a tag <sub>.
